# Test Task — Тарифы (Next.js + Tailwind)

Небольшое приложение на **Next.js (App Router)**: страница с выбором тарифа, промо-таймером и отображением скидки. Тарифы загружаются с удалённого API, по умолчанию выбирается “хит” (best) тариф, а UI корректно реагирует на окончание акции.

---

## Функциональность

- Загрузка и отображение **списка тарифов** с удалённого API
- **Выбор тарифа** по клику
- Автоматическая сортировка тарифа с флагом **`is_best`** (“Хит!”)
- **Промо-таймер (120 секунд)**:
  - во время промо отображается **скидка и промо-цена**
  - по окончании таймера промо **деактивируется**
  - последние ~30 секунд — визуальный срочный-режим
- **Чекбокс согласия** (обязательный) перед нажатием кнопки покупки
- Полностью **адаптивная верстка**

---

## Использованные технологии

- **Next.js (App Router)** — маршрутизация, серверные и клиентские компоненты
- **React** — хуки и управление состоянием
- **Tailwind CSS v4** — стилизация и адаптивная верстка
- **Context API** — глобальное состояние промо
- **clsx** — условное формирование `className`
- **next/font** — Google Fonts + локальные шрифты (`woff2`)
